# License Decision

Decision: Apache-2.0.

The project is open source under the Apache License 2.0.

## Rationale

Apache-2.0 is a permissive open-source license with an explicit patent grant. It is commonly used for developer tooling, infrastructure projects, and agent-related platform work.

The project owner chose Apache-2.0 over MIT because it remains permissive while providing a more complete license text and patent language.

## Practical Effect

Under Apache-2.0, users can use, modify, distribute, and contribute to the project under the license terms.

This aligns the project with a community-oriented driver ecosystem, where browser, Tauri, Electron, native GUI, and visual QA integrations can receive outside contributions.

## Release Status

- `LICENSE` contains the Apache License 2.0 text.
- Package metadata uses `Apache-2.0`.
- README license section says `Apache-2.0`.
- v0.1 release notes include `Apache-2.0`.
