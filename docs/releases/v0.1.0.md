# v0.1.0 Release Notes

Status: release candidate / draft.

License: Apache-2.0.

## Highlights

- Linux-first isolated Xvfb desktop sessions.
- Screenshot capture.
- Mouse and keyboard input.
- Window list and focus.
- Evidence artifacts.
- CLI interface.
- HTTP JSON API.
- MCP stdio server.
- Sample Vite GUI demo.
- Visual Annotation Handoff MVP.
- Annotation repair demo.

## Verified Smoke Commands

- `pnpm smoke:x11`
- `pnpm smoke:http`
- `pnpm smoke:mcp`
- `pnpm smoke:vite:http`
- `pnpm smoke:vite:mcp`
- `pnpm smoke:annotation-repair`

## Main Demo

```sh
pnpm smoke:annotation-repair
```

This demo starts a local Vite app, opens it in a browser inside Xvfb, captures a broken UI screenshot, creates a visual annotation handoff, saves crop and markdown evidence, captures an after-fix screenshot, and cleans up all related processes.

## Known Limitations

- Linux/X11/Xvfb-first.
- Coordinate-based fallback interactions.
- Browser semantic driver planned but not implemented.
- Tauri and Electron semantic drivers planned but not implemented.
- Rectangle-only annotation UI.
- HTTP is local-only and unauthenticated.
- Sessions are in-memory per server process.
- Visual repair smoke proves handoff and verification, not autonomous LLM repair.
