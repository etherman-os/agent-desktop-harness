# Sample Electron App

Minimal Electron app used by `smoke:electron-driver`.

It mirrors the sample Vite workflow with:

- an input placeholder: `Type a message`
- a `Save message` button
- an `Open details` button
- `Status: saved`
- `Details panel is open`

Run locally:

```sh
pnpm --filter @agent-desktop-harness/sample-electron-app dev
```
